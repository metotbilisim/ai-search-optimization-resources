# Answer Engine Optimization (AEO)

Answer Engine Optimization focuses on structuring content so that answer engines can retrieve clear, concise answers to user questions.

Typical AEO elements:

- FAQ sections
- Direct answers
- Semantic headings
- Entity references
- Structured data
