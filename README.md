# AI Search Optimization Resources

This repository contains references and educational resources about **AI Search Optimization (AISO)** created by Metot Bilişim.

AI Search Optimization is the process of optimizing websites so that artificial intelligence systems such as:

* ChatGPT
* Google AI Overviews
* Microsoft Copilot
* Perplexity AI
* Google Gemini

can discover, understand and cite them as trusted sources in generated answers.

## Main Guide

Comprehensive guide published by Metot Bilişim:

https://www.metotbilisim.com/ai-search-optimization/

This guide explains:

* What AI Search Optimization is
* How AI systems select sources
* AI citation signals
* Differences between SEO and AI Search
* Generative Engine Optimization (GEO)
* Answer Engine Optimization (AEO)
* Entity SEO
* Knowledge Graph Optimization
* Topical Authority Strategy
